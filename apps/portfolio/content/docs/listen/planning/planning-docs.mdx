## Loop

| Step | Rule |
| --- | --- |
| `1` | update these Listen planning docs before implementation |
| `2` | execute one logical task from `listen-surface-01` (or a successor phase) |
| `3` | verify `pnpm run build` and `pnpm run lint`; spot-check `/listen` in dev |
| `4` | mark tasks done in `task-registry.mdx` and refresh `state.mdx` |
