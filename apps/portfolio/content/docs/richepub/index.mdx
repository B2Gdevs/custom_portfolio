# RichEPub

RichEPub is a custom ebook container format (**.repub**) that supports React, Tailwind CSS, and npm dependencies. It is intended for web-based readers only and is not compatible with standard EPUB readers like Kindle or Apple Books.

## Documentation

- **[Format Specification](./format-spec)** – Container layout, `repub.json` schema, dependency semantics, and versioning.

## Related

- **[Book reader (EPUB)](/docs/books/reader)** – In-app reader loads `book.epub`; embeddable in docs via BookReaderEmbed.
- Standard **EPUB** is used for reading; `pnpm run build:books` produces .epub from repo-root `books/`. The **.repub** format is legacy; the portfolio no longer builds or serves .repub (Koodo/Kookit can still open it when wired).
