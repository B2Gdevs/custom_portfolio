# State

**Next action:** optional **Apps hub** skeleton when Payload is slow (roadmap annotate-03 remainder) or OCR language packs.

**Shipped (2026-04-07):** **Auto-redact** (`screenshot-annotate-04-04`) — **Redact emails / phones / both** runs word-level Tesseract, places solid rectangles; **camera constraints** (`screenshot-annotate-03-01`) keep pan/zoom on the screenshot.

**Shipped (2026-04-07):** **Text scan** — **Tesseract.js** runs in-browser on **Text scan**; modal spotlight + per-line copy + copy all ([requirements — text scan](/docs/screenshot-annotate/requirements#text-scan--spotlight-snipping-toolstyle--shipped); tasks `screenshot-annotate-04-01`–`03`).

**Shipped (2026-04-06):** **`pick`-phase paste** — **Clipboard API first** (`navigator.clipboard.read()`), then `clipboardData.files`, then `image/*` **DataTransferItem** + **`File` snapshot** before async load — fixes **blank canvas** after paste on Windows/Chrome when the naive `kind === 'file'` path returned nothing or stale blobs ([requirements — Clipboard and paste](/docs/screenshot-annotate/requirements#clipboard-and-paste); task `screenshot-annotate-01-04`).

**Shipped (annotate-02):** **Context menu** works while **geo/text** tools are active (removed upstream select-tool-only gate); **Reorder** submenu + **Delete** unchanged from tldraw defaults. **Style panel** scoped to stroke, fill, color, opacity, dash, size, fonts, text alignment, and geo variant — no arrow/line-only rows.

**Shipped (annotate-01):** Route **`/apps/screenshot-annotate`** — `ImagePicker` → **tldraw** `Tldraw` with custom **DefaultToolbar** (four tools) → **Export PNG** / preview screen; **`FALLBACK_SITE_APPS`** + seed; immersive shell via **`isImmersiveAppsRoute`** (no site footer).

**Resolved (product):**

- **Canvas:** **[tldraw](https://tldraw.dev/)** is the implementation base — [Clipboard](https://tldraw.dev/sdk-features/clipboard), editor APIs, export; no parallel canvas stack ([requirements](/docs/screenshot-annotate/requirements#implementation-stack--tldraw-authoritative)).
- **Data:** User screenshots **never** saved to Postgres, Supabase, Payload uploads, or S3 — **browser-only, short-lived**; see [requirements](/docs/screenshot-annotate/requirements#privacy-and-persistence-non-negotiable).
- **Chrome:** **Minimal** — immersive editor, essential controls + exit to `/apps`; portfolio direction is to tighten chrome for **Dialogue Forge and other `/apps` tools** over time (see [documentation cross-cutting queue](/docs/documentation/planning/state#cross-cutting-queue)).
- **Hide / redact:** **Solid** fill (or high-opacity) for v1; **blur** deferred to a later planning row.
- **OCR / spotlight / copy text:** **Planned** phase **annotate-04** (Snipping Tool Text Actions–style); engine **Tesseract.js** in a **Web Worker**, **lazy-loaded** when the user enables text actions ([requirements — planned feature](/docs/screenshot-annotate/requirements#planned-feature--text-scan--spotlight-snipping-toolstyle)); tasks **`screenshot-annotate-04-*`** in task registry.
- **v1 toolbar:** **Four tools only** — **Select**, **Hand**, **Rectangle/geo**, **Text** — not the full default tldraw strip ([requirements](/docs/screenshot-annotate/requirements#screenshot-annotate-v1-toolbar)).
- **Paste:** **Pick** phase = explicit clipboard handler before tldraw mounts; **annotate** phase = tldraw [Clipboard](https://tldraw.dev/sdk-features/clipboard) APIs for native behavior ([requirements — Clipboard and paste](/docs/screenshot-annotate/requirements#clipboard-and-paste)).
- **Export:** **Raster PNG only** in v1 ([requirements — Export format](/docs/screenshot-annotate/requirements#export-format)).

**Open questions:**
