# Kickoff: `global-auth-01`

| Field | Value |
| --- | --- |
| `goal` | Establish one shared auth and entitlement layer for gated site features before Books and Listen add backend-backed persistence or private media access. |
| `scope` | Payload auth-enabled users, tenant/org seed, owner seed, long-lived session policy, local auto-login for development, and shared helpers for permission checks. |

| `tests-required` | unit coverage for session parsing, owner seed behavior, and permission helpers; build/lint must pass. |
| `dependencies` | current Payload integration, existing RAG collections, and books/listen planners consuming the new policy instead of older lock systems. |
