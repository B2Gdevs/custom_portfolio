# Dialogue Forge

A visual node-based dialogue editor that exports to Yarn Spinner format for use in Unreal Engine.

## Try It

**[Open Dialogue Forge →](/apps/dialogue-forge)** — Graph, Yarn, and Play modes in one workspace.

## Features (shipped)

- **Graph** — React Flow canvas: NPC lines, player choice nodes, and conditional branches; connect edges and edit in the side panel
- **Yarn** — Text view of the same tree for quick reading and review
- **Play** — Step through the dialogue like a conversation
- **Memory flags** — Schema and runtime flags; open the flag manager from the graph toolbar when exposed
- **Yarn export** — Download a `.yarn` file for the Yarn Spinner plugin in Unreal (export is wired from the editor; graph mode includes the usual export affordances)
- **Persistence** — Dialogue and flags are saved in the browser (portfolio store backed by `localStorage`)

## Workflow

1. Open **`/apps/dialogue-forge`** and set the **title** in the header.
2. Use **Graph** to add nodes, connect them, and edit text and choices in the panel.
3. Switch to **Yarn** or **Play** to validate tone and flow.
4. Export to **Yarn** when you are ready for engine integration.

Keyboard shortcuts for undo/redo apply when focus is not inside an input (see [Editing Guide](/docs/dialogue-forge/editing-guide)).
