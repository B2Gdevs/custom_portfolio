# Task Registry

## Id shape (books section)

All ids here use **`<books>-<stream>-<phase>-<task>`** (see [Global planning guide](/docs/global/global-planning)). Streams in this registry:

| Stream | Meaning |
| --- | --- |
| `reader` | EPUB reading surfaces, annotations, read-page planning strip |
| `publishing` | Editions and which book EPUBs are live |
| `ai` | Site chat + RAG roadmap: **`books-ai-00`** = shipped CopilotKit (task `books-ai-00-01`); **`books-ai-01`** = forward Payload / `sqlite-vec` / ingest / citations queue |

Legacy-looking ids such as `books-ai-01-04` are **not** a separate scheme: **`ai`** is the stream name, same as **`reader`** in `books-reader-01-16`.
