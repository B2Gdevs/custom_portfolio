## Key Features

- ✅ **Modular Design** - Transplant to any React app
- ✅ **Real-Time Collaboration** - WebSocket + Y.js CRDT
- ✅ **Version Control** - Git-like branching and merging
- ✅ **Rich Text Editing** - TipTap with HTML output
- ✅ **Export** - EPUB and HTML export
- ✅ **Grimoire Aesthetic** - Dark, mystical book-like interface

## Architecture Highlights

- **Service Injection Pattern** - Components receive services via context
- **No Database Dependencies** - Component library is database-agnostic
- **Normalized Schema** - Books → Chapters → Sections
- **Lazy Loading** - Chapters loaded on-demand
- **Auto-Save** - Automatic saving with retry logic
